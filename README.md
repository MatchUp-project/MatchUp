팀원 별 프로젝트 소감
 
 유재린
이번에 시스템 아키텍처 페이지를 만드는 작업은 처음 해보는 경험이었지만, 실제 프로젝트의 구조를 하나로 정리해보면서 이해도가 훨씬 높아지는 걸 느낄 수 있었다. 각 기능이 어떤 흐름으로 연결되는지 눈에 보이게 정리되니까 프로젝트 전체가 하나의 서비스로써 어떻게 움직이는지 더 명확하게 와닿았고, 완성된 페이지를 보면서 개인적으로도 꽤 뿌듯했다. 처음 시도한 작업이었지만, 앞으로 문서화나 구조 설계할 때 좋은 기준이 될 것 같다는 점에서도 의미 있는 과정이었다.

 탁정훈

솔직히 지금 현재 우리의 능력으로 하기는 힘든 프로젝트였다고 생각한다. 아직 안 배운 언어들은 사용해야 했기 때문에 그렇게 느꼈다. 하지만 과연 같은 학년 대에 다른 학교 애들이 이런 경험을 할 수 있을까? 생각해보면 그 또한 아닌 거 같아서, 한 층 성장했다는 느낌이 들었다. 평소 개별의 언어를 사용하면서 잘 와닿지 않았던 부분이 이번 프로젝트를 통해서 각 언어들이 어떤 곳에 사용되고 어떤 역할을 하는 지 몸으로 느낄 수 있었다는 점이 이 프로젝트의 가장 큰 장점이라고 생각한다. 앞으로 팀 프로젝트 혹은 개별 프로젝트를 하는 데에 있어 이번 프로젝트가 준 베네핏을 가지고 좀 더 쉽게 임할 수 있을 거 같다.
